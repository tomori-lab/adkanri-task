# デプロイ手順（統合アプリ）

## 構成

| コンポーネント | ホスティング | URL例 |
|---------------|-------------|-------|
| 統合アプリ（依頼+タスク） | GitHub Pages | https://AXIS-AD.github.io/adkanri-task/ |
| 依頼フォーム | GAS（iframe埋め込み） | - |
| タスクダッシュボード | Render | https://xxx.onrender.com |

---

## 1. 統合アプリ（GitHub Pages）

1. https://github.com/AXIS-AD/adkanri-task を開く
2. **Settings** → **Pages**
3. **Source**: Deploy from a branch
4. **Branch**: `master` / **Folder**: `/docs`
5. **Save**

push するたびに自動デプロイされる。

---

## 2. タスクダッシュボード（Render）

1. [Render](https://render.com) にログイン
2. **New** → **Web Service**
3. リポジトリを接続（adkanri-task）
4. 設定:
   - **Root Directory**: `task-dashboard`
   - **Build Command**: `npm install`
   - **Start Command**: `npm start`
   - **Instance Type**: Free
5. **Environment** に以下を追加:
   - `CHATWORK_API_TOKEN`
   - `CHATWORK_MY_ACCOUNT_ID`
   - `CHATWORK_ROOM_1`, `CHATWORK_ROOM_2`
   - `CHATWORK_ROOM_AD` = `333632829`
   - `AD_MEMBER_IDS` = `10034061,9797164,5420288,10696465`
6. **Create Web Service**
7. デプロイ完了後、`https://xxx.onrender.com` のURLをコピー

---

## 3. config.js の設定

1. `docs/config.js` を開く
2. `DASHBOARD_URL` に Render のURLを設定:
   ```js
   var DASHBOARD_URL = 'https://adkanri-task-dashboard.onrender.com';
   ```
3. コミットして push（GitHub Pages が自動デプロイ）

---

## 4. 動作確認

1. https://AXIS-AD.github.io/adkanri-task/ を開く
2. **依頼する** タブ: GAS フォームが表示されること
3. **タスク一覧** タブ: 全タスク or 自分の依頼を表示
