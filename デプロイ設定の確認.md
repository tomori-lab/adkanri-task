# デプロイ設定の確認

「依頼者: AXISアドレスのみ利用可能です」と表示される場合、以下を確認してください。

## 1. デプロイを「ユーザーとして実行」にする

1. GAS エディタで **デプロイ** → **デプロイを管理**
2. 既存デプロイの **編集**（鉛筆アイコン）
3. **実行ユーザー** を **「自分」** ではなく **「アクセスしているユーザー」** に変更
4. **デプロイ** をクリック

→ これでログイン中のメールアドレス（xxx@axis-ads.co.jp や xxx@shibuya-ad.com）が取得され、依頼者名として自動設定されます。

## 2. 許可ドメイン

以下のドメインでログイン・依頼が可能です：

- `@axis-ads.co.jp`
- `@axis-hd.co.jp`
- `@shibuya-ad.com`

## 3. 依頼者名の自動認識

ログインしている Google アカウントのメールアドレスが、そのまま依頼者名としてチャットワークに記録されます。入力不要です。
